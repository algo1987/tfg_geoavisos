# tfg_geoavisos
Aplicación móvil con recordatorios de tareas basados en la geolocalización del usuario

## APK de prueba

En la carpeta `releases/` se incluye un APK de prueba de la aplicación:

`GeoAvisos-v1-debug.apk`

Esta versión permite instalar la aplicación en un dispositivo Android para comprobar su funcionamiento.
